# Toggl Focus — AI Onboarding Prototype

Interactive prototype for Toggl Focus AI onboarding, built with React, TypeScript, and Vite.

## Features

- AI setup flow: input → draft review → timer/calendar view
- Mock streaming LLM with rule-based draft generation
- Editable project/task tree before confirmation
- Calendar suggestions with accept/decline
- Toggl Focus timer shell with sidebar, calendar grid, and tasks panel

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Scripts

- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run preview` — preview production build
